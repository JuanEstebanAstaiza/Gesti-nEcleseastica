# Glosario
- Donación: aporte económico (diezmo, ofrenda, misiones, especial).
- Comprobante: archivo (PDF/JPG/PNG) que respalda una donación.
- Documento: registro en `documents` que vincula un archivo a donación/usuario/evento.
- Evento: actividad eclesiástica con posibles inscripciones y donaciones asociadas.
- JWT: token de autenticación para sesiones.
- Storage: carpeta o bucket donde se guardan archivos subidos.
